# Yoga Layout

Yoga Layout은 사용 편의성을 고려해 설계된 오픈 소스 레이아웃 엔진입니다. Yoga는 이미 널리 사용되는 오픈 소스 프레임 워크를 지원합니다. Yoga를 사용하면 엔지니어가 플랫폼 간에 보다 쉽게 ​​공동 작업 할 수 있는 간단하고 직관적인 레이아웃 API를 제공 할 수 있고, 부드럽고 유연한 UI 성능을 보장하기 위해 스레드의 레이아웃을 계산하는 것과 같은 흥미로운 기능을 제공합니다.
